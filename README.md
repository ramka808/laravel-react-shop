# Shop on laravel+react
## Test assigement for 'A La Carte'

#### 1 - Installation Steps

Run :   

1 - ```npm install```     
2 - ```composer install```   
3 - ```php artisan key:generate```    
4 - ```php artisan jwt:secret```    
5 - ```npm run dev``` 
7 -  ```php artisan migrate:fresh --seed ```
  
6 - ```php artisan serve```
